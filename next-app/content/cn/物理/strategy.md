# 高考物理抢分策略

## 第一阶段：必拿的“基础分”（预计 25-30分）
**特点：** 考点单一，逻辑直接，理解后极难丢分。

| 知识点 | 题型 | 核心得分要点 | 建议分值 |
| :--- | :--- | :--- | :--- |
| 原子物理/核物理 | 选择题 | 光电效应方程、能级跃迁（$h\nu = E_m - E_n$）、核衰变方程。 | 6分 |
| 几何光学/物理光学 | 选择题/填空 | 折射定律（$n = \sin i / \sin r$）、全反射、双缝干涉距离公式。 | 6分 |
| 机械振动与波动 | 选择题 | 波动图象与振动图象的转化（$v = \lambda f$），掌握“上坡下坡法”。 | 6分 |
| 热学（选修） | 选择/计算 | 理想气体状态方程 $\frac{PV}{T} = C$，热力学第一定律 $\Delta U = Q + W$。 | 10-15分 |

## 第二阶段：高回报的“套路分”（预计 25-30分）
**特点：** 题目看起来难，但物理过程非常固定，适合智力水平高的同学快速建模。

| 知识点 | 题型 | 核心模型与二级结论 | 建议分值 |
| :--- | :--- | :--- | :--- |
| 万有引力与航天 | 选择题 | 黄金代换： $GM = gR^2$；基本思路： 万有引力提供向心力（记熟“高低速大”规律）。 | 6分 |
| 静电场 | 选择题 | 等势面与电场线分布；电势能与做功关系；带电粒子在电场中的类平抛。 | 6分 |
| 磁场（圆周运动） | 选择/大题前两问 | 找圆心、求半径、定圆心角。 牢记 $R = \frac{mv}{qB}$，利用几何关系（勾股定理、三角函数）解题。 | 6-10分 |
| 力学实验 | 实验题 | 打点计时器求瞬时速度和加速度（逐差法）；控制变量法分析。 | 6分 |

## 第三阶段：硬骨头里的“过程分”（预计 15-20分）
**特点：** 这部分是压轴题，零基础很难拿满，但由于你理解力好，可以通过“方程堆砌”拿走 60% 的分数。

| 知识点 | 题型 | 抢分策略 | 预计拿分 |
| :--- | :--- | :--- | :--- |
| 动量与能量 | 大题/压轴题 | 不管题目多难，先写三大方程： 动能定理、动量守恒定理、能量守恒。 | 6-8分 |
| 电磁感应 | 计算大题 | 四步走： 1. 求电动势 $E=BLv$；2. 求电流 $I=E/R$；3. 求安培力 $F=BIL$；4. 根据牛顿定律或能量列方程。 | 6-10分 |
| 电学实验 | 实验题 | 记住测电阻的两种方法（内接/外接）；伏安法测电源电动势和内阻（$U-I$ 图象）。 | 8-10分 |
