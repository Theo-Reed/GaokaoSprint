# 高考冲刺 - 驾驶舱仪表盘

本面板包含 GaokaoSprint 系统的核心策略与状态监控。

## 模块导航

### 1. [学习计划表 (Schedule)](./study_schedule)
- **目标**：在 16 周内从零基础冲刺至 550 分。
- **核心内容**：
    - 四阶段时间轴：基础重筑、模块攻坚、真题饱和、精准打磨。
    - 每日 14 小时高强度时间片段划分。

### 2. [饮食营养方案 (Diet)](./diet_plan)
- **目标**：为高强度逻辑工作提供每小时 100-110kcal 的稳定大脑能量。
- **核心内容**：
    - 每日 2400-2500 kcal 总量，3+2 分餐模式。
    - 关键营养素：肌酸、高剂量复合维生素 B、叶黄素、牛磺酸等补给时间表。
    -  glycemic stability (血糖稳定性) 控制方法，避免“饭后困”。

### 3. [学习环境优化 (environment.md)](./environment.md)
- **目标**：创造“觉醒水平最高”的物理物理空间。
- **核心内容**：
    - 空气质量：CO² 监控与通风阈值设置。
    - 光照黑客：500-750 Lux 高照度设置与 90-90-90 人体工学办公姿势。
    - 急性干预：利用冷水刺激诱发“潜水反射”快速唤醒。

### 4. [疲劳干预机制 (fatigue_intervention.md)](./fatigue_intervention.md)
- **目标**：处理 14 小时学习期间的突发困倦与认知饱和。
- **核心内容**：
    - 三层响应模型：秒级生理黑客、分钟级代谢重塑、系统级根因诊断。
    - **AI 熔断机制**：针对突触饱和的强制保护方案，防止无效学习产出。
