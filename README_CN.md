# GaokaoSprint (中文项目简介)

[English (Primary)](./README.md) | [中文版 (Chinese)](./README_CN.md)

**GaokaoSprint** 是一个结合 AI 辅助技术的考前加速框架。

我们的目标是通过 4 个月的高强度生理与认知优化，助力基础薄弱者在极短时间内冲刺高考高分。14 小时的学习效率通过科学的补给、环境监控和及时的状态干预得到保障。

## 核心亮点
- **AI 辅助方案**：深度结合大模型，精确定向薄弱点。
- **生物控制论**：将人体视为系统，优化血糖、氧气和神经递质。
- **短期极速路径**：从零到五百五，专注高产出、高价值的复习内容。
