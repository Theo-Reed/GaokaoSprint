# GaokaoSprint (中文版)

[English (Primary)](./README.md) | [中文版 (Chinese)](./README_CN.md)

**GaokaoSprint** 是一个结合 AI 辅助技术的冲刺框架，旨在通过 4 个月的高强度优化，让零基础或基础薄弱者快速达到指定的高考分数目标。

## 核心理念：AI 驱动 + 生物优化
- **AI 赋能**：利用 AI 助理生成个性化练习、深度分析错题逻辑、并模拟考场阅卷，实现“精准打击”。
- **生物黑客**：通过精确的血糖控制（饮食）、二氧化碳监测（环境）及神经修复（疲劳干预）来提升认知负荷上限。
- **短期突击**：针对高产出比 (ROI) 的科目进行“饱和攻击”。
