# GaokaoSprint

A high-performance, short-term preparation framework for Gaokao (National College Entrance Examination), focusing on biological optimization, rigorous scheduling, and physiological intervention.

## Quick Start / 快速开始

- **English Version**: [README_en.md](./README_en.md)
- **中文版 (Chinese)**: [cn/README.md](./cn/README.md)

## Core Components / 核心组件

1. **Study Schedule / 学习计划**: [EN](./study_schedule_en.md) | [CN](./cn/study_schedule.md)
   - 4-month rapid progression from fundamentals to mock exams.
2. **Diet & Nutrition / 饮食营养**: [EN](./diet_plan_en.md) | [CN](./cn/diet_plan.md)
   - 2400-2500 kcal high-logic fuel with timed supplements.
3. **Environment Optimization / 环境优化**: [EN](./environment_en.md) | [CN](./cn/environment.md)
   - $CO_2$ monitoring, lighting, and thermal management.
4. **Fatigue Intervention / 疲劳干预**: [EN](./fatigue_intervention_en.md) | [CN](./cn/fatigue_intervention.md)
   - 3-tier physiological response to cognitive burnout and drowsiness.

## Philosophy

Maximize output by controlling the physical variables: glucose stability, synaptic recovery, and environmental stressors.
通过控制生理变量（血糖稳定、突触恢复、环境压力）来实现学习产出的最大化。
