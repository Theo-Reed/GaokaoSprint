# 学习环境（按重要性排序）

1. 空气质量 — 二氧化碳监控（最高优先级）

- 风险：密闭房间内 CO₂ 容易快速上升；>1000 ppm 会降低认知表现，约 1500 ppm 会导致昏沉、决策变慢与头痛。
- 目标：尽量维持 CO₂ < 1000 ppm。
- 操作：每 2 小时强制通风 10 分钟；如有 CO₂ 检测仪，读数超 1000 ppm 则立刻通风；作为短期应对可做深腹式呼吸。

2. 温度 — “冷头暖足”

- 目标：18–22°C（必要时空调设 ~22°C）。
- 逻辑：>24°C 会引发外周血管扩张，降低脑灌注并增加困意。
- 操作：保持头面周围空气凉爽、脚部保暖（厚袜），避免全身过热。

3. 光照 — 昼夜节律化设置

- 08:00–17:00（高效期）：5000–6500K，桌面照度 500–750 Lux，抑制褪黑素，提升分析能力。
- 18:00–22:00（巩固期）：3000–4000K，减少蓝光，准备夜间记忆巩固。
- 防眩光：光源侧前方布局，避免屏幕或书页反光；环境亮度应与 Mac M1 Pro 屏幕相近。

4. 人体工学 — 90-90-90 原则

- 姿势：肘、髋、膝近似 90°。
- 屏幕：屏幕上沿与眼平齐，使用支架或垫书提升 M1 Pro。
- 桌面：保持简洁，减少边缘注意力耗费。

5. 声学环境 — 稳定的掩蔽噪音

- 使用粉红噪音或褐噪来掩盖突发声，稳定背景声谱。
- 禁止带词音乐学习；歌词会占用语言处理资源，干扰阅读与理解。

6. 冷水刺激 — 快速唤醒工具

- 机制：面部冷刺激提升去甲肾上腺素，可能诱发潜水反射并通过血管收缩减轻面部浮肿，从而短时间提高警觉性。
- 实用：面部冷水拍脸 10–30 秒或短时凉水淋浴 30–60 秒可作为快速唤醒手段。心血管疾病者请先咨询医生，避免长时间强冷暴露。
